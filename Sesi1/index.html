Sesi 1

Praktek Pemprograman Web 1 akan membahas tentang cara menampilkan halaman web dengan menggunakan HTML, CSS dan Javascript. Mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan pemahaman tersebut kemudian menuliskan hasil kreasinya sendiri.

CSS

Cascading Style Sheet (CSS) merupakan standar untuk mengatur tampilan yang ada pada suatu berkas HTML. CSS bukan bahasa pemprograman tetapi merupakan bahasa deklaratif yang dinyatakan langsung untuk menyatakan suatu tampilan, tidak terdapat logika, percabangan, ataupun perulangan

Javascript

Javascript merupakan bahasa pemprograman yang berjalan disisi browser, disisi pengguna, atau di peramban bukan server
